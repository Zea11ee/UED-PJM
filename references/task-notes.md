# 任务接口与权限

## token 与权限
- user_access_token 必须是任务成员所在租户；否则 403/404。
- 401/99991677：token 过期，刷新或重新登录。

## 常用接口（task.v2）
- GET /task/v2/tasks/{guid}：详情（due/members/status）。
- 常见错误：403/1470403（调用者非成员或跨租户）、404（GUID 无效或跨租户）。

## 排查
1) 确认授权选择的租户正确。
2) 调用人是否在任务成员/清单共享范围内。
3) 若 Doc 显示关注人但 API 403，可能同名不同租户；核对 open_id。

## 快速指派准则
- 负责人必须 UED 部门；非 UED 需确认。
- 未确定时保持空/待确认，不要乱写。
